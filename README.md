# Mini Games Hub

Welcome to **Mini Games Hub** – a collection of fun, interactive browser games built with HTML, CSS, and JavaScript!

## What is this site?

Mini Games Hub is designed for quick entertainment and casual learning. Play a variety of mini games directly in your browser, with no downloads required.  
## Games Included

- **Alphabet Sequence Catcher**  
  Test your reflexes and memory by catching letters in the correct sequence.

- **Crazy Math Challenge**  
  Sharpen your math skills with rapid-fire questions and tricky twists.

- **Guess the Thing/Fruit**  
  Use clues to guess the correct item from multiple choices.

- **Color Match**  
  Quickly identify and click the correct color block (no names shown on colors for added challenge).

## How to run locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Yashwanth9898GitHub/mini-games-hub.git
   cd mini-games-hub
   ```

2. Open `index.html` in your browser.
